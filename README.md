# react-hook-text-overflow :potable_water:

A React hook to determine if text is overflowing in an element showing an ellipsis.

## Installation

Using `npm`:

```sh
npm install react-hook-text-overflow
```

Using `yarn`:

```sh
yarn add react-hook-text-overflow
```
